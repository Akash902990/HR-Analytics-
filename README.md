# HR Analytics Project

## Overview
This project analyzes employee attrition and workforce metrics using SQL and Excel. The project includes ETL processes, data transformation, analytical SQL queries, and dashboard reporting.

## Skills Demonstrated
- SQL ETL
- Data Cleaning
- Joins
- Views
- Subqueries
- Stored Procedures
- Excel Dashboarding
- KPI Reporting

## Dataset
- HR_1 Dataset
- HR_2 Dataset

## Key KPIs
- Employee Count
- Attrition Rate
- Department-wise Attrition
- Gender Distribution
- Job Satisfaction
- Employee Performance
